EXPLICATION DU CODE

Ci-dessous, j'explique le code de ce projet.

Le robot est codé en C++, grâce à l'utilisation d'une Arduino MegaADK et de deux moteurs, ainsi que d'une batterie 12V et d'un récepteur USB bluetooth et IR.

Le code ci-dessus nous permets de déplacer le robot en utilisant soit une Wiimote connectée via Bluetooth, soit une télécommande infrarouge connectée via le récepteur.

Le code inclus les bibliothèques puis les configure, ce qui nous permet d'utiliser les directions du moteurs et les broches de modulation de largeur de puissance comme sortie. Nous pouvons, ensuite, les utiliser pour déplacer le robot en avant, en arrière, à gauche, à droite et le faire s'arrêter.

LE PROJET LOCAL SÉCURISÉ

Ce projet est une poignée créée rendre votre maison ou vos locaux plus sûrs.

Ce projet est une poignée qui s'adapte à tout type de porte.

Il utilise un scanner de puces RFID, un clavier et une caméra pour assurer votre sécurité et celle de vos biens. Vous pouvez contrôler la poignée et tout le reste depuis un site web et une application dédiée.

EXPLICATIONS

Ci-dessous, je vais expliquer le projet de manière plus approfondie.

La porte utilise principalement deux cartes, un Raspberry Pi 3B+ chargé de la version ARM Server d'Ubuntu, ainsi que son système graphique basé sur Lubuntu.

Il utilise également un Arduino MegaADK avec une carte de lecture RFID et un clavier que nous avons fabriqués.

Le Raspberry Pi nous permettra de créer un serveur web et de l'héberger. Cela permettra à l'utilisateur d'utiliser le site web afin de se connecter à son profil pour utiliser la porte.
