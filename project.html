THE YEETBA V1

Also known as a vacuum robot,

The YEETBA V1 is a simple vacuum robot made to work efficiently along a user-specified area that can be defined inside the app itself.

At first, we had multiple ideas for the bot, such as making it as a military defense bot, a vacuum robot (our current idea), and also a cooking bot.

CODE EXPLANATION

Below, I will explain the code for this project.

The bot is coded in C++, through the use of an Arduino MegaADK and two motors, as well as a 12V battery and a bluetooth USB and an IR receiver.

The code above allows us to make the bot move simply by using either a Wiimote connected through Bluetooth, or a infrared remote connected via the receiver.

The code first includes libraries then sets them up, allowing us to use the motors's directions and power width modulation pins as an output. We can then use this to move the bot forwards, backwards, left, right and make it stop.

THE LOCAL SECURE PROJECT

This project is made as a new type of handle to make your house or supply rooms more safe.

This project is a handle that fits on any type of door.

It uses a RFID chip scanner, a keypad and a camera to keep yourself and what you own safe. You can control the handle and everything from a dedicated website and app.

EXPLANATION

Below, I will explain the project in further depth.

The door uses two cards primarily, a Raspberry Pi 3B+ loaded with Ubuntu's ARM Server version, as well as it's Lubuntu based graphics display.

It also uses an Arduino MegaADK with a RFID scanning board and a keypad which we made.

The Raspberry Pi will allow us to create a web server and host it. This will allow the user to use the website in order to log in to their profile to use the door.
